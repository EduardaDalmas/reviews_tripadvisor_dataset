*Avaliações de viagens do Leste Asiático*
Com o crescimento do setor de turismo e a expansão das plataformas de review, como TripAdvisor e Google Reviews, as opiniões dos viajantes se tornaram fontes valiosas de informação para novos usuários e para as próprias empresas do setor. Esses feedbacks são fundamentais para que empresas ajustem seus serviços e para que os futuros clientes façam escolhas mais informadas. Entretanto, o grande volume de reviews torna a análise manual impraticável, exigindo o uso de técnicas automáticas para processar e classificar as informações de forma rápida e precisa.
Neste projeto, apliquei técnicas de machine learning para analisar e classificar reviews de viagens com o intuito de automatizar a identificação do sentimento predominante. Ao categorizar as avaliações como positivas ou negativas, podemos fornecer uma visão geral sobre a satisfação dos clientes, destacando insights relevantes sobre os pontos fortes e fracos dos destinos ou serviços avaliados.
O objetivo deste projeto é desenvolver um modelo de classificação de sentimento utilizando reviews de viagens disponíveis no dataset "Travel Reviews". A classificação automática permitirá identificar se o conteúdo da review é positivo ou negativo, ajudando empresas de turismo e usuários a entenderem rapidamente a percepção geral dos destinos e serviços.
Para alcançar esse objetivo, seguiremos as etapas principais de um pipeline de machine learning, incluindo:
Análise Exploratório dos Dados: Explorar o dataset para entender suas características, identificar variáveis principais, verificar dados ausentes e realizar visualizações.
Pré-processamento dos Dados: Limpar e transformar o texto das reviews para facilitar a modelagem, incluindo a tokenização, remoção de palavras irrelevantes e conversão para uma representação numérica.
Desenvolvimento do Modelo: Selecionar e treinar um modelo adequado de machine learning para classificação
Avaliação do Modelo: Avaliar o desempenho do modelo com métricas como acurácia, matriz de confusão e análise de erros.
Este projeto visa não apenas implementar um modelo de classificação de sentimentos, mas também explorar como técnicas de NLP (Processamento de Linguagem Natural) e machine learning podem contribuir para melhorar a análise de feedbacks no setor de turismo.
Segundo as informações coletadas no site em que foi escolhido o dataset, este conjunto de dados é preenchido pelo rastreamento do TripAdvisor.com. São consideradas avaliações sobre destinos em 10 categorias mencionadas no Leste Asiático. Cada avaliação do viajante é mapeada como Excelente (4), Muito Bom (3), Médio (2), Ruim (1) e Péssimo (0) e a classificação média é usada em cada categoria por usuário.
Escolhi este dataset pelo meu interesse em viagens e por ter pouco conhecimento e experiência no assunto, me atrai bastante pelos dados disponibilizados e para conhecer melhor e pela experiência real de usuários sobre viagens e as coisas positivas e negativas que passaram.


*Sobre a análise exploratória:*
Neste estudo, foi realizada uma análise exploratória dos dados de feedback fornecidos pelos usuários em diversas categorias de interesse, como arte, clubes de dança, restaurantes e outros. A seguir, descrevo as análises realizadas e seus potenciais usos.
1. Distribuição das Notas nas Categorias
Para entender como as notas se distribuem nas diversas categorias avaliadas, foi gerado um histograma para cada categoria. Essa visualização permite identificar se as avaliações tendem a ser mais altas ou mais baixas, o que pode indicar a satisfação geral dos usuários em diferentes áreas. Por exemplo, se uma categoria como "feedback sobre galerias de arte" tiver uma distribuição concentrada em notas altas, isso sugere uma aceitação positiva, enquanto uma distribuição mais dispersa ou concentrada em notas baixas pode indicar áreas que precisam de melhorias. Essa análise é útil para priorizar áreas de investimento ou intervenção, além de ajudar na compreensão das preferências do público.
2. Média de Feedback por Categoria
Calculamos a média das avaliações em cada categoria e representamos esses valores em um gráfico de barras. Isso permite visualizar rapidamente quais áreas receberam avaliações mais altas ou mais baixas de forma comparativa. Ao entender quais categorias são mais valorizadas pelos usuários, é possível direcionar esforços para otimizar as áreas mais populares e, ao mesmo tempo, identificar categorias com baixa avaliação que podem precisar de ajustes ou mais atenção. Essa análise é importante para tomada de decisão em termos de alocação de recursos e estratégias de melhoria contínua.
3. Correlação entre as Categorias
A matriz de correlação foi utilizada para verificar como as diferentes categorias de feedback se relacionam entre si. Isso é relevante para identificar padrões ou grupos de categorias que podem estar ligados, como feedbacks positivos em arte podendo influenciar a avaliação de teatros ou restaurantes. Ao entender essas correlações, podemos identificar tendências, como a possibilidade de um usuário que gosta de arte também gostar de outros tipos de entretenimento, e usar essa informação para criar campanhas segmentadas ou personalizadas.
4. Relação entre Feedback de Galerias de Arte e Restaurantes
Um gráfico de dispersão foi gerado para observar a relação entre o feedback dado para galeria de arte e restaurantes. Essa análise pode revelar se usuários com altas avaliações para uma categoria tendem a avaliar bem a outra, ajudando a identificar preferências comuns entre os usuários. Esse tipo de análise pode ser útil para empresas ou organizações que queiram criar pacotes de produtos ou promoções cruzadas, ou até mesmo para recomendação de outros serviços de acordo com o histórico de avaliação do usuário.
5. Análise de Padrões e Agrupamentos (Clustering)
Aplicamos técnicas de clustering para identificar grupos de usuários com padrões semelhantes de avaliação. Utilizando o algoritmo K-means, dividimos os dados em três clusters baseados nos feedbacks das categorias. Essa análise é útil para segmentar usuários em grupos com características similares, permitindo a personalização de serviços ou ofertas. Por exemplo, pode-se criar campanhas focadas em grupos específicos, como um grupo de usuários com preferências por restaurantes e teatros, ou outro grupo mais interessado em arte e clubes de dança. O agrupamento pode fornecer insights valiosos para estratégias de marketing e melhorias nos serviços oferecidos.
6. Dispersão das Notas e Identificação de Outliers (Boxplot)
Foi realizada uma análise de dispersão das notas através de boxplots para observar a variabilidade das avaliações em cada categoria e identificar possíveis outliers. A presença de outliers pode indicar feedbacks extremos que merecem uma investigação mais aprofundada. Essa análise é útil para entender a consistência das avaliações e pode ser aplicada para detectar casos em que usuários têm opiniões divergentes, o que poderia ser investigado mais a fundo para melhorar a qualidade dos serviços ou para fornecer feedback individualizado aos usuários.
Utilidade das Análises
Essas análises têm várias aplicações práticas, especialmente para empresas ou organizações que desejam entender o comportamento e as preferências de seus usuários. Ao identificar padrões de avaliação, correlações entre categorias e segmentos de usuários com preferências semelhantes, as empresas podem tomar decisões informadas sobre como melhorar seus serviços, segmentar campanhas de marketing de maneira mais eficaz e otimizar a alocação de recursos.
Além disso, a detecção de outliers e padrões de dispersão pode ajudar a identificar áreas críticas que exigem ajustes ou mais atenção, enquanto o clustering pode fornecer insights valiosos sobre como agrupar usuários com base em seu comportamento e preferências. Com essas informações, é possível melhorar a experiência do usuário, personalizar ofertas e promover a fidelização do cliente.
